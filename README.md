# 1nce-device-integrator-sample

## Environment

### Hardware
- Raspberry Pi 3b+
- Grovepi+
- DHT11

### Software
- Python 3.5.3
- Raspbian For Robots (v9.13)

## Usage

Change your environment
~~~python
ENDPOINT = "127.0.0.1"
CLIENT_ID = "test-raspberrypi3b-01"
~~~

Excute below command
~~~python
python3 udp-iot-dht11.py
~~~
